Instrucciones para Configuración Directa

  1. Conecta tu dispositivo a la red WiFi "SistemaAcceso-XXXX"
  2. Una vez conectado, podrás configurar el ESP32 directamente sin necesidad de internet
  3. Haz clic en "Cargar Formulario ESP32" para comenzar
  4. Después de configurar, el ESP32 se reiniciará y se conectará a la red WiFi configurada
